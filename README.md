## 🌌 Concept

**「環境がクリエイティビティを規定する」**

`blender_theme` は、長時間の作業でも目が疲れにくく、かつ没入感を高めるために設計されたBlender用テーマのコレクションです。
視認性と美しさを兼ね備えたこれらのテーマは、あなたの創作活動を「作業」から「芸術」へと昇華させます。

---

## 🎨 Themes

収録されている厳選テーマ一覧です。

| テーマ名 | 特徴 |
| :--- | :--- |
| **Deep Forest** | 深い森をイメージした落ち着いたグリーンベースの配色。目に優しく、長時間の作業に最適です。 |
| **Night Storm** | 嵐の夜のようなダークブルーとグレーのコントラスト。クールで集中力を高めるプロ仕様。 |
| **Soft Pastel** | 柔らかいパステルカラーを採用した、明るく親しみやすいインターフェース。 |
| **My Modo** | 業界標準ツール「Modo」の配色を再現。移行ユーザーにも馴染みやすいデザイン。 |
| **Sky** | 澄み渡る空のような爽やかなブルー。開放感のある作業環境を提供します。 |
| **Old Concept** | クラシックなCGツールの雰囲気を漂わせる、ノスタルジックかつ実用的な配色。 |

## 🚀 Installation

1. このリポジトリをZIPとしてダウンロード、またはCloneします。
2. Blenderを開き、`Edit` > `Preferences` > `Themes` に移動します。
3. `Install...` ボタンをクリックし、ダウンロードしたXMLファイルを選択します。
4. プリセットからインストールしたテーマを選択して適用します。
